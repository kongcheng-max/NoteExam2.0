# BUG-035: V1.1前端代码丢失，仅剩MVP版本

**发现日期**: 2026-06-30
**优先级**: P0
**位置**: `app/frontend/src/`
**状态**: 待修复

## 问题描述

V1.1 阶段开发的前端功能代码（文件上传、知识点CRUD、试卷导出按钮、试题编辑等）全部丢失，当前前端仅包含 MVP 基础功能（粘贴文本笔记 + 生成试卷 + 查看试卷）。

## 根本原因

1. 研发部在开发 V1.1 前端时未执行 `git commit`
2. 修复 BUG-034（编码损坏）时执行 `git checkout` 恢复文件，将 `Home.jsx`、`ExamModal.jsx`、`api.js`、`vite.config.js` 恢复到了 MVP 初始版本
3. 所有 V1.1 前端改动因未提交而永久丢失

## 影响

- 文件上传（图片/PDF）功能不可用
- 知识点管理面板不可用
- 试卷导出（HTML/PDF）入口不可用
- 试题编辑/删除/重排序功能不可用
- 前端代理配置回退

## 修复建议

1. 重新开发 V1.1 前端功能，参考后端已有的 API（`/api/notes/upload`、`/api/knowledge-points`、`/api/exams/{id}/export` 等）
2. **每次完成一个功能模块立即 `git commit`**
3. 在 `vite.config.js` 中重新配置 `/api` 代理到 `localhost:8000`
