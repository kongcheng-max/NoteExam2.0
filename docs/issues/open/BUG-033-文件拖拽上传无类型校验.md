# BUG-033: 文件拖拽上传无类型校验

**发现日期**: 2026-06-29
**优先级**: P2
**位置**: `app/frontend/src/pages/Home.jsx` handleFileDrop
**状态**: 待修复

## 问题描述

拖拽上传的 `handleFileDrop` 接受任意文件类型，不校验 MIME 类型：

```javascript
const handleFileDrop = (e) => {
    e.preventDefault();
    const file = e.dataTransfer?.files?.[0];
    if (file) setSelectedFile(file);  // 不校验类型
};
```

## 影响

- 用户拖入 .docx/.txt/.mp4 等不支持的文件后，上传到后端才报错
- 体验差：应在前端提前拦截

## 修复建议

```javascript
const handleFileDrop = (e) => {
    e.preventDefault();
    const file = e.dataTransfer?.files?.[0];
    if (!file) return;
    const allowed = uploadMode === 'pdf'
      ? ['application/pdf']
      : ['image/jpeg', 'image/png'];
    if (!allowed.includes(file.type)) {
      showToast('Unsupported file type', 'error');
      return;
    }
    setSelectedFile(file);
};
```
