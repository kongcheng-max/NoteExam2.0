# BUG-026: 试卷导出使用英文标签

**发现日期**: 2026-06-29
**优先级**: P2
**位置**: `app/backend/services/export.py:6-12` TYPE_LABELS / DIFF_LABELS
**状态**: 待修复

## 问题描述

导出试卷中的题型和难度标签全部使用英文，与中文界面不一致：

```python
TYPE_LABELS = {
    "single_choice": "Single Choice",
    "multi_choice": "Multiple Choice",
    "fill_blank": "Fill in Blank",
    "true_false": "True/False",
    "short_answer": "Short Answer",
    "essay": "Essay",
}
DIFF_LABELS = {"basic": "Basic", "advanced": "Advanced", "challenge": "Challenge"}
```

## 影响

- 导出的试卷中标签为英文，与前端中文标签不统一
- 用户体验割裂

## 修复建议

```python
TYPE_LABELS = {
    "single_choice": "单选题",
    "multi_choice": "多选题",
    "fill_blank": "填空题",
    "true_false": "判断题",
    "short_answer": "简答题",
    "essay": "论述题",
}
DIFF_LABELS = {"basic": "基础", "advanced": "进阶", "challenge": "拔高"}
```
