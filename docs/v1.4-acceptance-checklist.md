> 验收标准：输入邮箱 + 密码 + 确认密码即可注册；密码至少 6 位；邮箱格式校验。

| # | 验收项 | ✓ |
|---|--------|---|
| 13 | 输入有效邮箱 + 6 位以上密码 + 确认密码一致，注册成功 | |
| 14 | 邮箱格式不合法时给出校验提示 | |
| 15 | 密码少于 6 位时给出提示 | |
