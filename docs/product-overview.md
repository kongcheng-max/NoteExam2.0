# NoteExam 产品概述

## 产品简介

**NoteExam** 是一款基于 AI 的智能出题工具——只需上传你的学习笔记，DeepSeek 大模型便会自动提取知识点，为你生成一份覆盖全面、题型丰富、难度分层的标准试卷，并附带答案与解析。

无论你是在备考冲刺的学生、需要出题测验的老师，还是自主学习某个领域的爱好者，NoteExam 都能帮你把零散的笔记瞬间变成高质量的测试题，让「学完就测」变得前所未有的简单。

---

## 核心价值

| 痛点 | NoteExam 的解决方案 |
|------|---------------------|
| **手工出题费时费力**：构思题目、设置选项、编排难度，一份试卷动辄数小时。 | 上传笔记，AI 自动完成知识提取与题目生成，数分钟内产出完整试卷。 |
| **知识覆盖不全面**：人工出题容易遗漏重点，或者偏重某一章节。 | DeepSeek 大模型全局分析笔记内容，确保每一个知识点都有对应题目。 |
| **难以自测效果**：做完题后缺少标准答案和解析，不知道哪里错了、为什么错。 | 每道题附带标准答案 + 关键要点解析，快速定位薄弱环节。 |

## 目标用户

### 🎓 学生

你正在备考期末考试、考研、考证或任何阶段性测试。手头有一堆笔记、课本标注、PPT 截图，但不知道从哪里开始复习。NoteExam 帮你一键生成针对性试卷，先测后学，效率翻倍。

### 👩‍🏫 教师

你需要为班级准备课堂小测、单元测试或期中试卷。传统出题流程繁琐且重复，NoteExam 让你只需上传课程笔记或教材章节，即可获得一份可直接使用或稍作修改的试卷，释放更多精力用于教学。

### 📚 自学者

你在自学编程、外语、经济学或任何领域，积累了大量笔记却缺少检验学习成果的手段。NoteExam 的 AI 出题能力让你随时随地进行自测，保持学习节奏。

---

## 使用场景

- **期末冲刺**：将整个学期的笔记一次性上传，生成综合模拟卷，查漏补缺。
- **课堂随测**：上传当堂课件，快速生成 5–10 分钟的课堂小测验。
- **章节自测**：学完一章后立刻测试，检验掌握程度。
- **错题回顾**：针对薄弱知识点重新生成试题，强化训练。
- **小组学习**：生成试卷后分享给同学，一起做题讨论。

## 核心功能一览

| 功能模块 | 说明 |
|----------|------|
| 📥 **笔记上传** | 支持拍照/截图（图片）、PDF 文件、以及直接粘贴文本，多种方式灵活导入。 |
| 🧠 **知识提取** | DeepSeek 大模型自动识别笔记中的核心知识点，构建知识图谱。 |
| 📝 **多样题型** | 单选、多选、填空、判断、简答、论述，六种题型覆盖各类测试需求。 |
| 📊 **难度分层** | 基础 / 进阶 / 拔高三档难度，适配不同水平，循序渐进。 |
| ✅ **答案与解析** | 每道题附带标准答案和关键要点解析，做一题懂一题。 |
| 📋 **试卷管理** | 支持生成、预览、导出试卷，方便打印或在线分发。 |

## 为什么选择 NoteExam？

| 对比维度 | 手动出题 | 传统题库 | **NoteExam** |
|----------|----------|----------|--------------|
| 题目来源 | 个人经验 | 固定题库 | **AI 根据你的笔记动态生成** |
| 知识覆盖 | 依赖出题人判断 | 题库覆盖范围有限 | **全笔记覆盖，无遗漏** |
| 出题效率 | 数小时一份 | 需从题库筛选 | **数分钟完成** |
| 个性化 | 可定制但耗时 | 通用试题，无针对性 | **完全基于你的学习材料** |
| 答案解析 | 需自行编写 | 通常仅有答案 | **AI 生成标准答案 + 关键解析** |
| 难度控制 | 主观判断 | 预定义难度 | **AI 智能分层** |

NoteExam 不是简单的题库替换，而是将你的学习笔记直接转化为个性化测试，让每一次测验都精准匹配你所学的内容。
